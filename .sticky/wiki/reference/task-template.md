General task

```
Create a task named “<title>” on board `<board-id>` with status `<status>`, priority `<n>`, tags `<[...]>`, and then set its content to this markdown:
Goal: …
Scope: …
Acceptance: …
```


Implement task

```
Implement the task "<title>" from `<board-id>`, update the task status and when it is done add summary of the work done to the task description
```
