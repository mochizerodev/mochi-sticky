Goal: Implement task templates for consistent task creation.

Scope:
- Create `.sticky/templates/tasks/` directory
- Support markdown templates with variable substitution
- Add `mochi-sticky template create <name>` command
- Add `mochi-sticky template list` command
- Add `--template <name>` flag to task creation
- Include built-in templates: bug, feature, meeting, review
- Allow custom template variables (e.g., {{author}}, {{date}})

Acceptance:
- Can create tasks from templates
- Templates support frontmatter and body content
- Variable substitution works correctly
- Built-in templates are useful and documented
- Custom templates can be created and shared
