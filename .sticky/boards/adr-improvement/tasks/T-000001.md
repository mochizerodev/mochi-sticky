Goal: Allow customization of ADR templates for team-specific needs.

Scope:
- Create `.sticky/templates/adrs/` directory
- Support custom ADR template files
- Add `mochi-sticky adr template create <name>` command
- Add `--template <name>` flag to `adr create`
- Include variables: {{id}}, {{title}}, {{date}}, {{author}}
- Provide multiple built-in templates (MADR, Y-statement, etc.)
- Validate template structure

Acceptance:
- Can create custom ADR templates
- Templates support variable substitution
- Built-in templates cover common formats
- Template validation prevents invalid ADRs
- Templates are shareable across teams
