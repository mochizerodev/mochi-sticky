Goal: Add table of contents auto-generation for wiki navigation.

Scope:
- Add `mochi-sticky wiki toc <page>` command to generate TOC
- Parse markdown headers (H2, H3, H4)
- Generate anchored links for each header
- Support --depth flag to control TOC levels
- Auto-update TOC marker in existing pages
- Add TOC to exported PDFs
- Integrate TOC into wiki index page

Acceptance:
- Can generate TOC for any wiki page
- TOC links are properly anchored
- TOC updates automatically when page changes
- Export formats include TOC
- TOC respects configured depth levels
