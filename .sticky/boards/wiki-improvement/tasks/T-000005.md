Goal: Implement comprehensive wiki search to quickly find content across all pages.

Scope:
- Implement `mochi-sticky wiki search <query>` command
- Search in page titles, content, tags, and section headings
- Add filters: --section, --tag, --status (published/draft)
- Support fuzzy matching and case-insensitive search
- Highlight matches in search results
- Add search integration to TUI wiki viewer
- Display results with context snippets
- Support regex patterns with --regex flag

Acceptance:
- Can search across all wiki pages in < 1 second for 100+ pages
- Results show page title, section, and matching context
- Filters work individually and in combination
- TUI shows search results with navigation
- Search is case-insensitive by default
- Regex search works for advanced queries
- Empty query shows helpful usage message
