# Security Policy

## Supported Versions

This project currently follows a rolling release model during `0.x`. Security updates will be provided for the most recent minor release.

## Reporting a Vulnerability

Please do not open a public issue for security reports.

Instead, email:
`mochizero0@gmail.com`

Include:
- A description of the issue
- Steps to reproduce
- Affected versions and environment details
- Any proof-of-concept or suggested remediation

We will acknowledge receipt within 7 days and aim to provide a fix or mitigation timeline as soon as practical.
